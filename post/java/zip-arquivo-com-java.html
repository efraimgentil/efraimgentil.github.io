Zip arquivo com Java

Zipando um arquivo ou pasta usando a ZIP util do Java

Postado por Efraim Gentil em 14 de Junho, 2021

Java possui uma série de utilitários "out of the box", um deles é a ZIP utils, com ela podemos comprimir ou descomprimir arquivos, vulgo zipar ou dezipar. Você pode checar todas as classes utilitárias de ZIP java no pacote java.util.zip.

Nesse post, veremos a implementação simples comprimindo um arquivo e uma pasta, e seus respectivos arquivos.

Primeiramente definimos o destino do arquivo usando o ZipOutputStream, em seguida para cada arquivo que iremos comprimir, devemos criar uma ZipEntry e adicioná-los no zipOutputStream. A chamada zipOutputStream.putNextEntry(...) indica que iremos começar a escrever o arquivo, e escrevemos o arquivo usando o método zipOutputStream.write(), que no nosso caso, estamos lendo e escrevendo todos os bytes. Quando não desejamos adicionar mais nenhum arquivo, fechamos o zip

Para comprimir uma pasta usamos os mesmos métodos, mas precisamos percorrer todos os arquivos dentro da pasta. O Java fornece um utilitário para isso, Files.walkFileTree. Utilizando esse método utilitário, fornecemos um FileVisitor, e podemos manipular cada arquivo visitador da forma que acharmos melhor.

Files.walkFileTree ira percorrer todos os arquivos, incluindo as pastas e arquivos dentro das pastas, todos serão comprimidos.

O método buildEntry está garantindo a mesma estrutura hierárquica de pastas e arquivos. Ao criar a ZipEntry, é possível definir a hierarquia.

Exemplo: Para comprimir um arquivo X.txt na pasta Y dentro do arquivo ZIP, temos ZipEntry("Y/X.txt")
