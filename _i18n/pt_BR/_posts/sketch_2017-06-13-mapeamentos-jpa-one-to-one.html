Nessa serie de artigos veremos os mapeamentos de entidades mais comuns usando anotacoes do JPA 2, os CascadeType, quando usalos e quando deixalos de lado, Eager e Lazy fething. Nos exemplos usaremos JUnit com Spring Test para rodar nossos testes e fazer nossas asserções.

Para criar e configurar o projeto usaremos a ferrament "https://start.spring.io/", basta selecionar JPA e H2 como dependencias e entao clicar em gerar o projeto.

Vamos iniciar com os relacionamentos um para um bi-direcional

{% highlight java %} @Entity public class User { @Id @GeneratedValue(strategy = GenerationType.AUTO) private Long id; @Column(length = 20, nullable = false) private String username; @OneToOne(cascade = CascadeType.ALL) @JoinColumn(name = "user_profile_id") private UserProfile userProfile; } {% endhighlight %}
